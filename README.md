# 概要
これはURLを通じて名刺をweb上に公開できる車好きに向けたアプリです \
自分の乗っている車、乗っていた車、欲しい車を公開することでコミニュティ内の会話をより良いものにします \
また、そこから車の詳細情報を見ることができます。
